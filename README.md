Installation process-
cd Quiz
npm install
npm run dev

How to play?
1-Click Start Quiz button  to begin quiz.
2️- Answer multiple-choice questions .
3️- Get instant feedback (whether the answer is correct or not).
4️- View your final score and Try Again.
